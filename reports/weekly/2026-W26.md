# 本周雷达 — 2026-W26

## 一句话总览
AI 机会转向“可控、可省、可审计”。

## AI Agent 的运行时治理与观测

**数据支撑**  
- Heron：把自己定位成 “Wireshark for AI Agents”，用 eBPF 做被动观测 [1179833]  
- Polygraph：让 AI agents 跨 repo 看到上下文并保持 session memory [1179782]  
- Agent Arena：公开比较 AI agents 表现，说明“怎么评估 agent”开始变成刚需 [1173517]  
- 2k 人攻击 AI assistant 的复盘，prompt injection / red-team 仍是生产化短板 [48681687]

**为什么这是机会**  
Agent 已经从 demo 进入真实工作流，但企业最缺的不是“再来一个 agent”，而是知道它做了什么、为什么做、何时越权、失败后如何复盘。现有可观测工具偏传统 API / infra，对 agent 的工具调用、上下文漂移、权限边界、记忆污染支持不够。

**与他的 SaaS 背景的契合点**  
如果他熟悉 B2B SaaS 的权限、审计日志、团队协作、客户成功，这是强契合：可以买给 CTO / Head of Eng / AI platform team。需要补的新能力是 agent runtime、LLM tracing、prompt injection 攻防和开发者体验。

**立即可做的小动作**  
- 访谈 8 个正在用 Claude Code / Cursor / Codex 的工程团队，问“agent 出错时你怎么定位责任”。  
- 做一个只支持 GitHub repo + Claude Code log 的轻量审计面板：工具调用链、文件改动、风险操作标红。

## LLM 成本与模型路由层

**数据支撑**  
- Smart model routing 直接接入 Claude、Codex、Cursor，HN 有 110 条讨论 [48688700]  
- Wayfinder Router：在本地 LLM 与 hosted LLM 之间做确定性路由 [48704373]  
- “Why current LLM costs are not sustainable” 引发 192 条讨论 [48683588]  
- DeepSeek DSpark speculative decoding 讨论热度高，说明推理效率仍是核心瓶颈 [48696585]

**为什么这是机会**  
AI coding / agent 使用量上来后，团队会第一次认真看 token 成本、延迟和隐私。大模型“全量调用”太贵，本地模型“全量替代”又不稳；缝隙在于按任务、风险、上下文自动选择模型，并给管理者可解释的成本账单。

**与他的 SaaS 背景的契合点**  
SaaS 创始人天然懂定价、用量计费、团队预算、权限和报表，可以把“模型路由”包装成 CFO / CTO 都能理解的成本控制产品。难点是要做出足够低摩擦的集成，而不是又一个开发者要维护的代理服务。

**立即可做的小动作**  
- 找 5 个每月 AI API / coding agent 花费超过 $1k 的小团队，拿真实账单看成本结构。  
- 原型先做 “Cursor / Claude Code 用量导入 + 节省建议”，不急着做完整路由器。

## 公司上下文记忆：从知识库到可行动层

**数据支撑**  
- Atlas：主张“每个 AI 工具都应该知道你的公司怎么运转” [1176131]  
- ClickUp Brain²：一个知道整个公司并能行动的 AI [1176406]  
- OpenKnowledge：AI-first Obsidian / Notion 替代，接入 Claude / Codex，本周 HN 170 条讨论 [48675435]  
- note.md：把本地 notes 和 research docs 变成 local LLM memory [1179055]

**为什么这是机会**  
多数企业不是缺聊天框，而是缺可信、持续更新、可授权的公司上下文层。现在每个 AI 工具都各自接入文档、Slack、CRM、工单，重复、过期、权限混乱；机会在做“上下文基础设施”，让不同 agent 共用同一套事实、权限和引用。

**与他的 SaaS 背景的契合点**  
这很适合有 SaaS 经验的人：连接器、权限模型、团队 onboarding、搜索体验、企业销售都能复用。需要新增能力是 RAG 质量评估、上下文版本控制、引用可信度和数据治理。

**立即可做的小动作**  
- 访谈 6 个 50-300 人 SaaS 公司，问“新人、销售、客服现在怎么确认公司事实”。  
- 做一个窄场景 MVP：只同步 Notion + Linear + Slack，回答“某功能为什么这样做、谁决定的、当前状态”。

## 反直觉的 1 个观察

本周最强信号不是“更强模型”，而是“模型访问受限”。GPT-5.6 / Mythos 的政府审批、可信组织放行、区域替代模型等讨论热度极高 [48690101] [48692995] [48697958]。这意味着未来企业 AI 采购会更像云安全和合规采购：谁能用、在哪里用、数据能不能出境、审计怎么留痕，都会变成产品机会。
